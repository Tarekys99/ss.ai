# Footer System

هذا المجلد يحتوي على ملفات Footer المشتركة لجميع صفحات الموقع.

## الملفات

- **footer.html**: Footer للصفحات الداخلية (pages/)
- **footer-home.html**: Footer للصفحة الرئيسية (index.html)

## كيفية الاستخدام

### في الصفحات الداخلية (pages/)
```html
<!-- استبدل footer الكامل بـ -->
<div id="footer-placeholder"></div>

<!-- وأضف في نهاية الصفحة قبل </body> -->
<script src="../js/load-footer.js" type="text/javascript"></script>
```

### في الصفحة الرئيسية (index.html)
```html
<!-- استبدل footer الكامل بـ -->
<div id="footer-placeholder"></div>

<!-- وأضف في نهاية الصفحة قبل </body> -->
<script src="js/load-footer.js" type="text/javascript"></script>
```

## التعديلات

لتعديل Footer في جميع الصفحات:
1. عدل الملف المناسب (footer.html أو footer-home.html)
2. احفظ التغييرات
3. سيتم تطبيق التغييرات تلقائياً على جميع الصفحات
