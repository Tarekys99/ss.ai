## 1. React Developer

### الوصف:
انضم إلى فريق الواجهات الأمامية لبناء تطبيقات ويب حديثة ومتجاوبة باستخدام React

### المتطلبات الأساسية:
✅ خبرة 3+ سنوات مع React.js
✅ معرفة قوية بـ JavaScript (ES6+)
✅ خبرة مع Redux أو Context API
✅ إتقان HTML5, CSS3, والتصميم المتجاوب
✅ معرفة بـ RESTful APIs
✅ خبرة مع Git version control
✅ فهم جيد لمبادئ UI/UX

### مهارات إضافية (Nice to Have):
- TypeScript
- Next.js framework
- الاختبار مع Jest/React Testing Library

### الأيقونة:
`icon-screen-desktop` - شاشة سطح المكتب

## 2. Flutter Mobile Developer

### الوصف:
بناء تطبيقات موبايل جميلة متعددة المنصات باستخدام Flutter framework

### المتطلبات الأساسية:
✅ خبرة 2+ سنوات مع Flutter
✅ معرفة قوية بلغة Dart
✅ خبرة مع إدارة الحالة (Provider, Bloc, Riverpod)
✅ فهم معمارية تطبيقات الموبايل (MVVM, Clean Architecture)
✅ خبرة مع REST APIs و JSON
✅ معرفة بمنصات iOS و Android
✅ خبرة مع Firebase أو خدمات backend مشابهة

### مهارات إضافية (Nice to Have):
- تطبيقات منشورة على App Store/Play Store
- خبرة في تطوير iOS/Android الأصلي
- معرفة بـ CI/CD لتطبيقات الموبايل

### الأيقونة:
`icon-screen-smartphone` - هاتف ذكي

### التمييز:
هذه الوظيفة مميزة بـ `pricing-active` (خلفية زرقاء)
